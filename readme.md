# 📊 Sales Data Analysis (Kaggle Project)

## 📌 Overview
This project analyzes a real-world retail dataset (Supermart Grocery Sales) to extract meaningful business insights using Python and Pandas.

---

## 🛠️ Tech Stack
- Python
- Pandas
- Matplotlib

---

## 📊 Key Analysis Performed
- Data Cleaning (handled missing values & duplicates)
- Category-wise Sales Analysis
- Sub-category Performance Analysis
- Monthly Sales Trend Analysis

---

## 🔍 Key Insights
- Highest sales were generated from **Eggs, Meat & Fish category**
- Peak sales occurred during **November and December**
- Certain sub-categories like **Health Drinks** performed consistently well

---

## 📈 Features
- Real-world dataset (Kaggle)
- Data preprocessing & transformation
- Business insights extraction
- Visualization of trends

---

## 🚀 Conclusion
This project demonstrates practical data analysis skills including data cleaning, aggregation, and visualization to derive business insights.
